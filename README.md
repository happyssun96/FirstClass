## 💡 Git 기본 규칙[rules]
1. 본인 파트 이외의 다른 담당자의 파트를 수정할 경우는 꼭 허락을 맡고 어느 파트의 어디를 수정하는지 알려주고 작업 수행
2. Git Push 하기전에 꼭 톡방에 이야기를 하고 작업 수행
3. 브랜치를 만들면 [기능 또는 업무명] / [작업자] (ex. LoginRegister/YBH)
